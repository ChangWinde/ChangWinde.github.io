Latex配置

Posted on 2018-03-04   |  

Windows

对于很多程序猿们来说，电脑上安装的东西太多了，这是我们想要追求更小更精简而同时也想完成我们的目标。此时Sublime+Latex就可以完美解决这个小问题

配置过程

  • 下载并安装所需的软件：

Sublime Text 3下载地址

MiKTeX 下载地址

Sumatra PDF 下载地址

  • 安装LatexTools：

    在Sublime中按 ctrl+shift+p，将会出现控制面板，然后再输入install，选择 Package Control: install package。进入库后，搜索LaTeXTools，直接回车即可安装。首先需要安装Package Control 比较简单，这里我们跳过这一步

  • 配置LatexTools：

    重启Sublime，配置文件，调出 Package Control，搜索LatexTools:Reset user settings to default并回车即可。这一步实际上在User目录下创建了名为LaTeXTools.sublime-settings的配置文件。在preferences->browse packages

进入User文件夹后找到LaTeXTools.sublime-settings文件，搜索关键字texpath 进行修改

修改为自己的Miktex的目录 类似上图所示。
